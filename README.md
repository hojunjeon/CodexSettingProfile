# CodexSettingProfile

재사용 가능한 Codex 작업 환경 프로필 저장소입니다.

## Profiles

- `profile1/`: 성능은 높게 유지하면서 토큰 사용량을 줄이기 위한 Codex + LazyCodex/OmO + ponytail + trihead 세팅입니다.

이 저장소에는 API 키, GitHub 인증 토큰, keyring, 개인 런타임 경로 같은 비밀값은 포함하지 않습니다. 새 PC나 새 워크스페이스에서는 `profile1/readme.md`를 먼저 읽고 설치 스크립트와 검증 스크립트를 순서대로 실행하면 됩니다.
